Musical Instruments Atlas provides a paged explorer with horizontal sliding. Current application version uses only portrait mode. An explorer page describe an instrument with picture, sample instrument audio, short description and facts. There is also a filtering mechanism by instruments category or spreading countries and a list for quick instrument lookup.
