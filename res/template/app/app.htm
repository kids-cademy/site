android system permissions

Full Network Access
Application needs to download atlas content from @string/site-name servers. Also needs to keep your device in sync.
View Network Connections
Perform download an sync only when connected on Wi-Fi, in order to avoid mobile data connections that can be charged.
Storage and Photos/Media/Files
Application needs to write content on device storage. Application content is stored into application private space.
